1. Generate two identical PDF images of an uppercase O using the files `raster.tex` and `vector.tex`.

2. Compile both with `xelatex` to get tow outputs of tight, cropped PDFs of the letter O:

```
xelatex raster.tex
xelatex vector.tex
```

3. Convert the file PDF to SVG:
`pdftocairo -svg vector.pdf vector.svg`

4. Convert the PDF to PNG:
`pdftocairo -png raster.pdf raster.png`

5. Remove white background in raster.png using `convert` from Imagemagick:

`convert raster.png -transparent white raster-transparent.png`

6. Copy over these two images from `auxiliary` to `images` so:

```
cp vector.svg ../images/vector.svg
cp raster-transparent.png ../images/raster.png
```
